This submission contains two major elements.

Firstly, the Dockerfile is the Dockerfile as used in the investigation. This Dockerfile was adapted, and can be readapted, to suit any of the discusssed exploits by making the changes detailed in the thesis.

Secondly, the GonnaCry directory contains all the adapted sourcecode of the GonnaCry ransomware. The original ransomware can be found at https://github.com/tarcisio-marinho/GonnaCry. The adaptations made are detailed in the thesis.
